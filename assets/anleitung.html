Suchfunktion:

Chip-Suche: weltweit
Tattoo-Suche: ifta intern
ID-Suche: ifta intern


Chip-Suche:

Tragen Sie in die erste Zeile den 15-stelligen Transponder-Code ein und klicken Sie auf die Lupe, um den Suchvorgang zu speichern und zu starten. Klicken Sie auf Speicher-Button, so wird der Chip ebenfalls gespeichert, die Suche jedoch (noch) nicht gestartet.
Suchergebnis: In der Suchliste werden alle Datenbanken aufgelistet, die durchsucht wurden. Ist das Tier bei ifta registriert, wird das Ergebnis oben grün angezeigt. Wurde das Tier in einer anderen Datenbank gefunden, können Sie durch klick auf diese weitere Informationen einholen.


Tattoo-Suche:

Tragen Sie das vollständige Tattoo oder erkennbare Teile für linkes, rechtes oder beide Ohren in die Felder ein und klicken Sie auf die Lupe, um den Suchvorgang zu starten. Klicken Sie auf Speicher-Button, so wird das Tattoo ebenfalls gespeichert, die Suche jedoch (noch) nicht gestartet.
Suchergebnis: Das Ergebnis wird in einer Tabelle dargestellt. Klicken Sie auf einen Eintrag, um nähere Informationen zu dem Tier zu erhalten.


ID-Suche:

Tragen Sie die achtstellige ifta-ID in das Feld ein und klicken Sie auf die Lupe, um den Suchvorgang zu speichern und zu starten. Klicken Sie auf Speicher-Button, so wird die ifta-ID ebenfalls gespeichert, die Suche jedoch (noch) nicht gestartet.
Suchergebnis: Das Ergebnis wird direkt dargestellt.



Tipp:

Sollten Sie als Vermittler zwischen Finder und Halter fungieren (z.B. Tierheim, Feuerwehr, Polizei…), so können Sie in die unteren Felder zwei Telefonnummern des Finders notieren sowie einen Kommentar verfassen, der z.B. den Fundort oder weitere Informationen enthalten kann.

Chip-Analyse:

Tragen Sie den 15-stelligen Transponder-Code ein. Sie können dann einsehen, aus welchem Land und von welchem Hersteller der Chip stammt.


Tier Registrierung:

Registrieren Sie ihr Tier direkt über Smartphone/Tablet bei ifta. Tierärzte können wie gewohnt ihre Kundennummer eintragen.


Log-In:

Nicht angemeldete Benutzer sehen die vollständigen Tierdaten sowie die Telefonnummern der Halter. Registrierstellen können sich mit Ihren üblichen Zugangangsdaten anmelden, um vollständige Einsicht in die Halterdaten zu bekommen.


Suchverlauf:

Jede Suchanfrage wird im Suchverlauf aufgelistet. Leere Felder bedeuten, dass die entsprechenden Daten nicht vorhanden sind. Mit einem Klick auf einen Eintrag wird ein Menü geöffnet.
Eintrag bearbeiten:
Telefonnummern des Finders und Kommentar können editiert werden.
Status ändern:
Ändern Sie den Status der Anfrage. Konnten Sie beispielsweise den Halter nicht erreichen, so können Sie dies mit „Rückruf“ vermerken.
Anrufen (Feld 3 bis 7):
Rufen Sie Halter oder Finder direkt an. Ist ein Feld leer, so ist keine Telefonnummer vorhanden.
Suche starten:
Suche wird mit der entsprechenden Anfrage noch einmal gestartet
Eintrag löschen:
Löscht den Eintrag (z.B. wenn der Fall erledigt wurde)
